## Installation & usage

1. Clone the repo
3. [Setup Open AI](https://platform.openai.com/docs/quickstart)
4. Place api key as `OPENAI_KEY` and `OPENAI_PROJECT_ID` in your `.env`
5. `pnpm start` command parameters by index
- [2] source direectory,
- [3] requested language
- [4] `.env` path
6. Run `pnpm start`
